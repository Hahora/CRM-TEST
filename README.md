# CRM My Sclad

CRM-система для управления визитами, клиентами и продажами. Включает расписание примерочных, аналитику, рассылки и систему тикетов.

## Стек

- **Vue 3** + TypeScript + Vite
- **Pinia** — управление состоянием
- **Vue Router 4** — маршрутизация
- **Tailwind CSS 4** — стили
- **AG-Grid** — таблицы данных
- **Chart.js** + vue-chartjs — графики
- **Lucide Vue** — иконки
- **date-fns** — работа с датами

## Модули

| Модуль | Описание |
|---|---|
| **Дашборд** | Статистика продаж, посещений, топ клиентов, быстрые действия |
| **Посещения** | Расписание примерочных по дням и филиалам, карточки визитов |
| **Клиенты** | База клиентов, создание, карточка клиента |
| **Продажи** | Список сделок с фильтрами и статистикой |
| **Отчёты** | Аналитика по продажам, менеджерам, посещениям, конверсии |
| **Рассылки** | Email / Telegram / MAX-рассылки |
| **Тикеты** | Система поддержки с чатом |

## Запуск

```bash
# Установить зависимости
npm install

# Запустить dev-сервер (http://localhost:5173)
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Переменные окружения

Скопируй `.env.example` в `.env` и заполни:

```env
VITE_API_BASE_URL=http://localhost:3000   # URL бэкенда
VITE_API_TIMEOUT=10000                    # Таймаут запросов (мс)
VITE_APP_TITLE=CRM System
VITE_DEBUG_MODE=false
```
